## Overview

Creates a new public client in the system. This endpoint is public and doesn't require authentication.

<Note>
  **No Authentication Required**
  
  This is a public endpoint. Clients are typically created automatically when booking appointments via `POST /api/citas`, but this endpoint allows manual client creation.
</Note>

## Request Body

<ParamField body="nombres" type="string" required>
  First name(s) of the client
</ParamField>

<ParamField body="apellidos" type="string">
  Last name(s) of the client (optional)
</ParamField>

<ParamField body="telefono" type="string">
  Phone number (optional)
</ParamField>

<ParamField body="email" type="string">
  Email address (optional)
</ParamField>

## Response

<ResponseField name="message" type="string">
  Success message: "Cliente creado"
</ResponseField>

<ResponseField name="cliente" type="object">
  The created client object
  
  <Expandable title="cliente fields">
    <ResponseField name="id" type="number">
      Auto-generated client ID
    </ResponseField>
    <ResponseField name="nombres" type="string">
      First name(s)
    </ResponseField>
    <ResponseField name="apellidos" type="string">
      Last name(s)
    </ResponseField>
    <ResponseField name="telefono" type="string">
      Phone number
    </ResponseField>
    <ResponseField name="email" type="string">
      Email address
    </ResponseField>
    <ResponseField name="created_at" type="string">
      Creation timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash cURL
curl -X POST http://localhost:3000/api/clientes \
  -H "Content-Type: application/json" \
  -d '{
    "nombres": "Pedro",
    "apellidos": "Martínez",
    "telefono": "+5551234567",
    "email": "pedro@example.com"
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost:3000/api/clientes', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    nombres: 'Pedro',
    apellidos: 'Martínez',
    telefono: '+5551234567',
    email: 'pedro@example.com'
  })
});
const data = await response.json();
```

## Example Response

```json 201 Created
{
  "message": "Cliente creado",
  "cliente": {
    "id": 42,
    "nombres": "Pedro",
    "apellidos": "Martínez",
    "telefono": "+5551234567",
    "email": "pedro@example.com",
    "created_at": "2024-03-05T15:30:00.000Z"
  }
}
```

## Error Responses

<Accordion title="400 Bad Request">
  ```json
  {
    "message": "Error al crear cliente"
  }
  ```
  
  Invalid data was provided (e.g., missing required `nombres` field).
</Accordion>

## Use Cases

- Pre-registering clients before appointment booking
- Importing client data from external systems
