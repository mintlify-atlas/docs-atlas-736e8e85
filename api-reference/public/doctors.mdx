---
title: 'List Public Doctors'
api: 'GET /api/medicos/public'
description: 'Retrieve a list of all active doctors'
---

This endpoint returns a list of all active doctors available for appointments. Only doctors with `activo: true` status are included in the response.

## Authentication

No authentication required - this is a public endpoint.

## Query Parameters

No query parameters required.

## Response

Returns an array of doctor objects.

<ResponseField name="doctors" type="array">
  Array of active doctor objects
  
  <ResponseField name="[].id" type="number">
    Doctor ID
  </ResponseField>
  
  <ResponseField name="[].persona_id" type="number">
    Associated person ID
  </ResponseField>
  
  <ResponseField name="[].especialidad_id" type="number">
    Specialty ID
  </ResponseField>
  
  <ResponseField name="[].email" type="string">
    Doctor's email address
  </ResponseField>
  
  <ResponseField name="[].colegiatura" type="string">
    Doctor's professional license number
  </ResponseField>
  
  <ResponseField name="[].activo" type="boolean">
    Active status (always true for this endpoint)
  </ResponseField>
  
  <ResponseField name="[].persona" type="object">
    Personal information of the doctor
    
    <ResponseField name="persona.nombres" type="string">
      Doctor's first name(s)
    </ResponseField>
    
    <ResponseField name="persona.apellidos" type="string">
      Doctor's last name(s)
    </ResponseField>
    
    <ResponseField name="persona.telefono" type="string">
      Doctor's phone number
    </ResponseField>
  </ResponseField>
  
  <ResponseField name="[].especialidad" type="object">
    Specialty information
    
    <ResponseField name="especialidad.nombre" type="string">
      Specialty name
    </ResponseField>
  </ResponseField>
</ResponseField>

## Error Responses

<ResponseField name="500 Internal Server Error" type="object">
  Returned when a server error occurs
  
  ```json
  {
    "message": "Error al listar médicos"
  }
  ```
</ResponseField>

```bash
curl https://api.example.com/api/medicos/public
```
