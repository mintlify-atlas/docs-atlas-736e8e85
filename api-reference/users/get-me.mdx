## Overview

Returns the complete profile of the currently authenticated user, including their associated person information.

<Warning>
  **Authentication Required**
  
  This endpoint requires a valid JWT token from Supabase Auth. The user must have either `admin` or `asistente` role.
</Warning>

## Authentication

Include the JWT token in the Authorization header:

```
Authorization: Bearer <supabase-jwt-token>
```

## Response

<ResponseField name="id" type="string">
  The Supabase authentication UUID
</ResponseField>

<ResponseField name="persona_id" type="number">
  ID of the associated person record (nullable)
</ResponseField>

<ResponseField name="role" type="string">
  User's role: `admin` or `asistente`
</ResponseField>

<ResponseField name="created_at" type="string">
  Timestamp when the user was created
</ResponseField>

<ResponseField name="persona" type="object">
  Associated person data
  
  <Expandable title="persona fields">
    <ResponseField name="id" type="number">
      Person ID
    </ResponseField>
    <ResponseField name="nombres" type="string">
      First name(s)
    </ResponseField>
    <ResponseField name="apellidos" type="string">
      Last name(s)
    </ResponseField>
    <ResponseField name="telefono" type="string">
      Phone number (nullable)
    </ResponseField>
    <ResponseField name="dni" type="string">
      National ID number (nullable)
    </ResponseField>
    <ResponseField name="foto_url" type="string">
      Profile photo URL (nullable)
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash cURL
curl -X GET http://localhost:3000/api/usuarios/me \
  -H "Authorization: Bearer YOUR_SUPABASE_JWT_TOKEN"
```

```javascript JavaScript
const response = await fetch('http://localhost:3000/api/usuarios/me', {
  headers: {
    'Authorization': `Bearer ${supabaseToken}`
  }
});
const profile = await response.json();
```

## Error Responses

<Accordion title="401 Unauthorized">
  ```json
  {
    "message": "Token requerido"
  }
  ```
  
  No authorization token was provided.
</Accordion>

<Accordion title="404 Not Found">
  ```json
  {
    "message": "Usuario no encontrado"
  }
  ```
  
  The authenticated user doesn't have a record in the system.
