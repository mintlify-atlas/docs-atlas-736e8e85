---

## Overview

Returns detailed information about a specific user, including their associated person data.

<Warning>
  **Admin Only**
  
  This endpoint requires authentication and the `admin` role.
</Warning>

## Path Parameters

<ParamField path="id" type="string" required>
  The Supabase authentication UUID of the user
</ParamField>

## Response

<ResponseField name="id" type="string">
  The Supabase authentication UUID
</ResponseField>

<ResponseField name="persona_id" type="number">
  ID of the associated person record (nullable)
</ResponseField>

<ResponseField name="role" type="string">
  User's role: `admin` or `asistente`
</ResponseField>

<ResponseField name="created_at" type="string">
  Timestamp when the user was created
</ResponseField>

<ResponseField name="persona" type="object">
  Associated person data (nullable)
  
  <Expandable title="persona fields">
    <ResponseField name="id" type="number">
      Person ID
    </ResponseField>
    <ResponseField name="nombres" type="string">
      First name(s)
    </ResponseField>
    <ResponseField name="apellidos" type="string">
      Last name(s)
    </ResponseField>
    <ResponseField name="telefono" type="string">
      Phone number
    </ResponseField>
    <ResponseField name="dni" type="string">
      National ID number
    </ResponseField>
    <ResponseField name="foto_url" type="string">
      Profile photo URL
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash cURL
curl -X GET http://localhost:3000/api/usuarios/a1b2c3d4-e5f6-7890-abcd-ef1234567890 \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN"
```

```javascript JavaScript
const response = await fetch(
  'http://localhost:3000/api/usuarios/a1b2c3d4-e5f6-7890-abcd-ef1234567890',
  {
    headers: {
      'Authorization': `Bearer ${adminToken}`
    }
  }
);
const user = await response.json();
```
